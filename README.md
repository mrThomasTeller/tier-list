# Tier List

Минималистичное веб-приложение для составления тир-листов (S, A, B, C, D). Один HTML-файл, без зависимостей и сборки.

> Это приложение полностью сделано с помощью [Claude Code](https://claude.com/claude-code) на модели **Claude Opus 4.7**.

**Демо:** https://mrThomasTeller.github.io/tier-list/

## Возможности

- Пять тиров: **S / A / B / C / D** + пул неотсортированных элементов
- Перемещение элементов:
  - Drag-and-drop на десктопе
  - Кнопки `S A B C D ↓` на каждом элементе (работают везде, включая мобильный)
- Кнопка **×** на каждом элементе — удаление
- Добавление элементов:
  - Поодиночке через кнопку **+**
  - Списком через **Импорт** (textarea, по одному элементу на строку, режимы «Добавить» / «Заменить»)
- **Автоподстановка emoji** для каждого элемента через OpenAI (опционально)
- **Экспорт** в Markdown с кнопкой «Копировать»
- **Сброс** распределения по тирам
- Три кнопки соотношения панелей: **30/70**, **50/50**, **70/30**
- Состояние сохраняется в `localStorage` (элементы, их тиры, выбранное соотношение)
- Автоматическая светлая/тёмная тема по системным настройкам (`prefers-color-scheme`)
- Адаптивная мобильная вёрстка:
  - Главный экран не прокручивается, прокручиваются только панели
  - Тулбар внизу экрана (с учётом safe-area)
  - Элементы в две строки (название сверху, кнопки снизу — растянутые на палец)
  - Элементы в тирах вплотную друг к другу для максимальной плотности

## Автоподстановка emoji

В тулбаре есть кнопка-шестерёнка (настройки): туда можно ввести OpenAI API-ключ и включить галочку «Подставлять emoji автоматически».

После этого:

- При **импорте** списка — один батч-запрос к `gpt-4o-mini` подбирает подходящий emoji для каждого названия и подставляет его префиксом (`🎬 Властелин колец`).
- При **добавлении одного элемента** через кнопку **+** — то же самое для одного имени.

Детали:

- Ключ хранится только в `localStorage` твоего браузера (под отдельным ключом, не в файле основного состояния) и отправляется напрямую в OpenAI с фронтенда. Никакого бэкенда нет.
- Если ключ не задан или галочка выключена — функция просто не активируется, импорт/добавление работают как обычно.
- Если запрос упал (нет сети, неверный ключ, превышен лимит) — элемент(ы) добавятся без emoji, в консоль выводится warning.

⚠️ На публичном демо (GitHub Pages) ключ не предзагружен — каждый посетитель должен ввести свой собственный.

## Запуск локально

Просто открой `index.html` в браузере. Никакой сборки не требуется.

```sh
open index.html
```
